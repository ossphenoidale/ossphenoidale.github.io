NOEDIkOM

Project Description:

In Lower Austria, there are many architectural and artistic treasures, like listed Monuments. Art in public places or historical places, which are, for different reasons, not accessible or well known.

This project aims to digitalize gems throughout Lower Austria and make them accessible to everyone in a digital way. This is accomplished through an application with the possibility of looking at sculpture in a 3D-Environment, or walking through museums or municipals using 360° tours.

Technical Deatils:

Accomplished is all of this through the Unity Engine. For the VR-Framework we used the BNG Framework VRIF. With WebGL and WebXR it is possible to take a look at it with a Webbrowser and even use VR through the web browser.

My part was to make a base build and port this to WebGL. Additionally, I managed the server and the server connection between the server and the application. I was also responsible for most of the bug fixing.
